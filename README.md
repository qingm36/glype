#### 介绍
基于PHP的Web代理glype汉化美化版v1.4.15（墨涩美化版），Glype Web代理是一个用 PHP 编写的基于 Web 的代理脚本，用来作为 web 的代理服务器，可帮你请求目标URL，并生成相应的页面，无需修改浏览器配置即可访问其他网站。
